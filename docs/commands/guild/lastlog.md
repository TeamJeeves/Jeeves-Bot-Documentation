# LastLog Command

Once you setup your Guild, this command can quickly grab the link from WarcraftLogs for the last log you uploaded! 


**Aliases:** `lastlog`

**Available in DM:** No

**Can Restrict to a channel:** Yes

**Global Support:** Yes

**Examples**

* `!lastlog` - Fetch the last log for your guild

**Developer Thoughts**
